pub unsafe extern "C" fn json_rpc_respond(
    ptr: u32, 
    len: u32, 
    chain_index: u32, 
    user_data: u32
)

Client is emitting a response to a previous JSON-RPC request sent using json_rpc_send. Also used to send subscriptions notifications.

The response or notification is a UTF-8 string found in the memory of the WebAssembly virtual machine at offset ptr and with length len. chain_index is the chain that the request was made to. user_data is the value that was passed to json_rpc_send.
