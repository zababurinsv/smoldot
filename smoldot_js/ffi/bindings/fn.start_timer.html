Function smoldot_js::ffi::bindings::start_timer[−][src]

pub unsafe extern "C" fn start_timer(id: u32, milliseconds: f64)

After at least milliseconds milliseconds have passed, must call timer_finished with the id passed as parameter.

When timer_finished is called, the value of monotonic_clock_ms must have increased by at least the given number of milliseconds.

If milliseconds is 0, timer_finished should be called as soon as possible.
