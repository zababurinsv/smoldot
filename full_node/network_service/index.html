Module full_node::network_service[−][src]

Background network service.

The NetworkService manages background tasks dedicated to connecting to other nodes. Importantly, its design is oriented towards the particular use case of the full node.

The NetworkService spawns one background task (using the Config::tasks_executor) for each active TCP socket, plus one for each TCP listening socket. Messages are exchanged between the service and these background tasks.
